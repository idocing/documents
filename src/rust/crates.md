# crates

## download crate
```
wget -O pkg_name-version.crate https://crates.io/api/v1/crates/pkg_name/version/download
wget -O pkg_name-version.crate https://static.crates.io/crates/pkg_name/pkg_name-version.crate
```
> for example
```
wget -O serde-1.0.0.crate https://crates.io/api/v1/crates/serde/1.0.0/download
wget -O serde-1.0.0.crate https://static.crates.io/crates/serde/serde-1.0.0.crate
```
